Bienvenue sur Axe Trx,
votre application privée pour gérer et initier vos transactions bancaires.

Axe Trx est une application innovante conçue pour vous offrir un contrôle total sur vos transactions bancaires.
Notre plateforme permet aux utilisateurs de s'inscrire et d'initier des transactions. Les identifiants et les transactions sont enregistrés dans une base de données SQL, organisée en deux tableaux distincts.

L'application utilise un modèle d'intelligence artificielle, basé sur la technologie de machine learning, pour prédire approximativement le jour et l'heure auxquels une transaction en attente sera approuvée.

Que vous souhaitiez initier une nouvelle transaction ou annuler une transaction existante, notre plateforme conviviale et sécurisée vous permet de le faire en toute simplicité.
