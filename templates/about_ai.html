À propos de l'IA

Introduction

Bienvenue sur la page "À propos de l'IA" de notre application. Ici, nous vous proposons des insights sur le modèle d'Intelligence Artificielle utilisé pour prédire les temps d'approbation des transactions.

Aperçu du Modèle d'IA

Notre application utilise un modèle de Régression Linéaire entraîné sur des données de transactions pour prédire le temps d'approbation des transactions financières. Le modèle prend en compte le type de transaction et le montant comme caractéristiques d'entrée pour estimer le temps requis pour l'approbation.

Formation et Évaluation

Le modèle d'IA est formé en utilisant des données historiques de transactions stockées dans notre base de données. Voici un aperçu du processus de formation :

  • Collecte des Données : Les données de transaction sont extraites de notre base de données, en excluant les transactions en attente.
  • Prétraitement des Données : Les données de transaction sont prétraitées, y compris l'encodage one-hot des types de transactions, la mise à l'échelle des caractéristiques numériques et l'imputation des valeurs manquantes.
  • Entraînement du Modèle : Un modèle de Régression Linéaire est formé en utilisant les données prétraitées pour apprendre la relation entre les caractéristiques des transactions et les temps d'approbation.
  • Évaluation du Modèle : La performance du modèle est évaluée à l'aide des métriques suivantes :
    • Erreur Quadratique Moyenne (EQM) : Mesure de la moyenne des carrés des écarts entre les prédictions du modèle et les valeurs réelles des données.
    • Erreur Quadratique Moyenne Racine (EQMR) : Racine carrée de l'Erreur Quadratique Moyenne (EQM), indiquant l'écart moyen entre les prédictions du modèle et les valeurs réelles, exprimée dans la même unité que la variable cible.
    • Coefficient de Détermination (R2) : Mesure statistique indiquant dans quelle mesure les variations de la variable dépendante sont expliquées par les variations des variables indépendantes dans le modèle. Plus proche de 1, meilleure est l'explication.

Capacités Prédictives

Une fois formé, le modèle d'IA peut prédire le temps d'approbation des nouvelles transactions en fonction de leur type et de leur montant. Les prédictions sont faites en temps réel, permettant aux utilisateurs d'estimer quand leurs transactions seront probablement approuvées.

Application dans Notre Plateforme

Dans notre plateforme, ce modèle d'IA améliore l'expérience utilisateur en fournissant des temps d'approbation estimés pour les transactions initiées par les utilisateurs. Cela aide à gérer les attentes et à améliorer la planification des transactions.
